Overview

This page documents the current state of my homelab and home network. It includes the hardware list, the network layout, and the core services. Some sections are placeholders while I finish wiring up monitoring and dashboards.

Core Hardware

HP DL380 Gen9

  • Role: NAS + game servers (one part of the overall setup)
  • CPU: 2x Intel Xeon E5-2683 v4 (16 cores / 32 threads each)
  • Memory: 128 GB RAM
  • Boot drive: T-Force 512 GB 2.5" SATA SSD
  • Storage: 12x Seagate ST6000NM0195 6 TB
  • NICs: HP Ethernet 1Gb 4-port 331i Adapter; HP Ethernet 10Gb 2-port 560FLR-SFP+ Adapter; HP Ethernet 1Gb 4-port 366T Adapter
  • HBAs: Smart Array P440ar Controller; Smart HBA H241; HP 12G SAS Expander Card

UniFi Cloud Gateway Ultra (UCG-Ultra)

  • Role: WAN gateway, IDS/IPS, UniFi Network controller, VPN tunneling, and VLAN routing
  • Processor: Quad-core ARM Cortex-A53 at 1.5 GHz
  • System memory: 3 GB
  • On-board storage: 16 GB
  • Default WAN ports: (1) 2.5 GbE RJ45
  • Max WAN port count: 4
  • IDS/IPS throughput: 1 Gbps
  • Power method: USB Type-C (5V DC/3A, adapter included)
  • Max power consumption: 6.2 W
  • Supported voltage range: 100-240V AC
  • Dimensions: 141.8 x 127.6 x 30 mm (5.6 x 5 x 1.2 in)
  • Weight: 520 g (1.1 lb)
  • Other features: Multi-WAN load balancing, 0.96" LCM status display, supports 30+ UniFi devices and 300+ clients
